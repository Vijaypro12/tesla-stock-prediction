# 🚀 Tesla Stock Price Prediction using Deep Learning

## 📌 Overview
This project predicts Tesla stock closing prices using deep learning models (SimpleRNN and LSTM).

## 🎯 Features
- Time-series forecasting using RNN & LSTM
- Data preprocessing and EDA
- Multi-step prediction (1-day, 5-day, 10-day)
- Streamlit web app for visualization

## 🧠 Models Used
- SimpleRNN
- LSTM

## 📊 Results
- LSTM outperformed SimpleRNN
- Evaluated using Mean Squared Error (MSE)

## 🛠 Tech Stack
Python, Pandas, NumPy, TensorFlow, Scikit-learn, Streamlit

## 🚀 How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app/app.py
